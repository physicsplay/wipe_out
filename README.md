# Guardians of Our Waters · Field Op

**Field Op** is an interactive educational game created for the eTwinning project **Guardians of Our Waters**.

The game invites students to become water guardians. They investigate environmental scenarios, read evidence, answer questions, make decisions, and learn how rivers, lakes, seas, and local water sources can be protected from pollution.

## Play the game

You can play the game here:

**https://physicsplay.github.io/Field_Op/**

## What the game is about

Students work in teams and face different water-related missions, such as:

- testing water quality,
- identifying pollution sources,
- responding to environmental emergencies,
- choosing responsible community actions,
- thinking about local and cross-border water protection.

The game is designed for teenagers and can be used in class, during an eTwinning online meeting, or as part of an environmental awareness activity.

## Game modes

The game includes two modes:

### One Device

Teams play on the same device and pass the screen from team to team.

### Multiple Devices

Teams can play from different devices using a room code. This is useful for partner schools playing together online.

## Languages

The game supports multiple languages, including:

- English
- Bulgarian
- Greek
- Spanish
- Italian

## Educational aims

The game helps students practise:

- scientific thinking,
- environmental awareness,
- teamwork,
- digital literacy,
- responsible citizenship,
- decision-making based on evidence.

## Project connection

This game is part of the **Guardians of Our Waters** project, which focuses on water quality, water conservation, international collaboration, and student action for cleaner and safer waters.

## How to use this repository

Open the game through the GitHub Pages link above, or download the repository and open `index.html` in a modern browser.

No installation is required.
